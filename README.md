##### I - Liga/Desliga ilumiacao
##### O - Liga/Desliga orbitas
##### P - Liga/Desliga plano orbital
##### WASD - Movimenta a camera

## Para instalar dependencias necessárias do jogo:

## Instruções para compilar:
######   make run
